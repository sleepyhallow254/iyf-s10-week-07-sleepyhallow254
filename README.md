## Project Description
This is a Todo List application built using React and Vite.  
The app allows users to add, delete, and mark tasks as completed. It also includes filtering options and saves data in the browser using localStorage so that tasks remain even after refreshing the page.

---

## Technologies Used
- HTML5  
- CSS3  
- JavaScript (ES6+)  
- React (useState, useEffect)  
- Vite  
- LocalStorage API  

---

## Features
- Add new tasks  
- Mark tasks as completed  
- Delete tasks  
- Filter tasks (All / Active / Completed)  
- Clear completed tasks  
- Persistent storage using localStorage  
- Responsive and modern UI design  

---

## How to Run

1. Clone the repository:
```bash
git clone https://github.com/sleepyhallow254/todo-react-app.git
```

2. Navigate into the project folder:
```bash
cd todo-react-app
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

5. Open in browser:
```
http://localhost:5173
```
